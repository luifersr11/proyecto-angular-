# Angular Reactive Stock Form

A simple Angular app to practice **Reactive Forms**. Lets you create and edit stock items with live validation.

## Features
- **Reactive Form** using `FormBuilder`, `FormGroup`, `Validators`.
- **Real-time validation**: required fields, minLength, min value.
- **Dynamic errors**: shows messages only when fields are touched & invalid.
- **Create/Edit mode**: empty form for new items, pre-filled for editing.
- **Standalone Component**: modern Angular style.
